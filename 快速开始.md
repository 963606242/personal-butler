# 快速开始

完整说明见 [README.md](README.md)，此处为最简步骤。

## 1. 安装依赖

```bash
cd personal-butler
npm install
```

> 安装 `better-sqlite3` 可能需要编译环境：Python 3、node-gyp；Windows 需 Visual Studio Build Tools。

## 2. 开发模式

```bash
npm run dev
```

会同时启动：

- Vite 开发服务器：<http://localhost:3000>
- Electron 窗口（或可在浏览器中访问上述地址）

## 3. 构建

```bash
npm run build
```

产物在 `dist-electron/`（Windows 默认生成 NSIS 安装包）。

---

- **配置**：API Key、主题等请在应用内 **设置** 页配置，或参考 [.env.example](.env.example) 使用环境变量。
- **数据位置与外部访问**：见 [docs/data-storage.md](docs/data-storage.md)。
- **本地 API（供脚本 / AI 调用）**：见 [docs/api-bridge.md](docs/api-bridge.md)。
